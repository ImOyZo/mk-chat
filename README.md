# MK-Chat

Simple and stupid real-time chat web app.


## Hosting

Clone the project

```bash
  git clone https://github.com/ImOyZo/mk-chat.git
```

Go to the project directory

```bash
  cd mk-chat
```

Config .env file
```bash
  cp .env.example .env && nano .env
```

Run app

```bash
  sudo docker compose up
```

Access the app

```bash
  http://localhost:3000
```
